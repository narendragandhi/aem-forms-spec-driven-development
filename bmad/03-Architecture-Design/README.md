# 03 - Architecture Design

This phase is driven by the "Architect Agent" of the BMAD framework. It focuses on defining the technical architecture of the AEM solution, ensuring scalability, performance, and maintainability.

## Artifacts in this Phase

- **[system-architecture.md](system-architecture.md)**: High-level overview of the AEM as a Cloud Service architecture, including service tiers, CDN configuration, CI/CD pipeline, and integration patterns.
- **[component-design.md](component-design.md)**: Detailed technical specifications for AEM components, including Sling Model implementations, dialog definitions, and HTL templates.
- **[dispatcher-rules.md](dispatcher-rules.md)**: Dispatcher configuration for caching strategies, security filters, URL rewriting, and cache invalidation rules.
