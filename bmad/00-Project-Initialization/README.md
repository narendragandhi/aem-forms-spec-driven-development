# 00 - Project Initialization

This phase is about setting up the foundation for the AEM project. It involves creating the AEM project from the archetype, setting up the development environment, and configuring source control.

## Artifacts in this Phase

- **aem-project-archetype-config.md**: This document outlines the configuration used to generate the AEM project from the Adobe-provided archetype. It includes the archetype version, and the parameters used for project creation.
